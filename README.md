# BasinFlow
 
## 🌊 BasinFlow -> 
### BasinFlow é um projeto voltado à engenharia e organização de dados hidrológicos, com foco em séries históricas pluviométricas.

## 🎯 Objetivo -> 

### * Desenvolver um fluxo estruturado de processamento de dados capaz de:
### * Realizar ingestão de dados brutos
### * Tratar valores inconsistentes e falhas
### * Padronizar estruturas de dados
### * Aplicar métricas de confiabilidade
### * Disponibilizar os dados para análise e visualização

## ⚙️ Tecnologias Utilizadas ->

### Python ,Pandas, Numpy e Tkinter.

## 📊 Aplicação ->

### O projeto é aplicado a bases hidrológicas reais, contribuindo para organização e validação de dados utilizados em estudos ambientais e de recursos hídricos. O projeto conta com uma interface gráfica desenvolvida em Tkinter, permitindo ao usuário configurar parâmetros, selecionar arquivos de entrada e executar o fluxo completo de processamento e validação de dados hidrológicos.
